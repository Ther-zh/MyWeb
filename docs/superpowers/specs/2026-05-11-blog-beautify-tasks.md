# 知帆博客美化 - 实施任务清单

## 项目概述
将知帆博客从当前简约风格升级为星空宇宙沉浸式体验，包含粒子背景、打字机动画、毛玻璃卡片等科技感元素。

---

## 任务列表

### 阶段一：样式基础重塑

- [ ] **任务 1.1**: 重构 CSS 变量系统
  - 定义星空宇宙配色方案（深蓝、紫色、星光蓝）
  - 定义毛玻璃效果变量
  - 定义发光效果变量
  - 定义动画时长和缓动函数变量

- [ ] **任务 1.2**: 重构全局样式
  - 深蓝紫色渐变背景
  - 重置基础样式（字体、间距、颜色）
  - 毛玻璃效果基础类
  - 发光效果基础类
