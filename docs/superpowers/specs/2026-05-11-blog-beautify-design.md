# 知帆博客美化 - 设计规格文档

## 为什么需要这个项目

用户的博客需要从当前的简约风格升级为星空宇宙沉浸式体验，增强科技感和视觉冲击力，同时保持内容的可读性和专业性。

## 设计愿景

打造一个让访客沉浸其中的技术星空，每个页面都是一片独特的宇宙空间。通过粒子效果、打字机动画、毛玻璃卡片等元素，营造出前沿科技与宇宙浩瀚相结合的独特氛围。

## 核心变更清单

## 功能需求

### 需求：粒子星空背景系统
网站全局背景应显示动态星空效果。

#### 场景：正常浏览
- **WHEN** 用户打开任意页面
- **THEN** 显示深蓝紫色渐变背景
- **AND** 背景中有漂浮的星点粒子
- **AND** 偶尔有流星划过效果
- **AND** 鼠标移动时产生光晕跟随效果

#### 场景：性能考虑
- **WHEN** 检测到低性能设备或移动端
- **THEN** 减少粒子数量
- **AND** 禁用流星效果
- **AND** 保持基本星空效果

---

### 需求：打字机英雄标题
首页英雄区域的标题应以打字机效果逐字显示。

#### 场景：页面加载
- **WHEN** 首页加载完成
- **THEN** 标题"Hi，我是知帆"逐字显示
- **AND** 光标在末尾闪烁
- **AND** 副标题淡入显示

---

### 需求：毛玻璃卡片效果
所有卡片组件应呈现毛玻璃质感。

#### 场景：卡片展示
- **WHEN** 卡片渲染在页面上
- **THEN** 背景显示半透明磨砂效果
- **AND** 边框带有微妙的光晕
- **AND** 悬浮时发光增强

#### 场景：悬浮效果
- **WHEN** 用户鼠标悬浮在卡片上
- **THEN** 卡片轻微上浮（translateY）
- **AND** 发光效果增强
- **AND** 阴影加深
- **AND** 平滑过渡 0.3s

---

### 需求：平滑过渡动画
页面元素应以优雅的动画方式展示。

#### 场景：页面加载
- **WHEN** 页面内容加载
- **THEN** 元素从下方淡入
- **AND** 依次延迟显示（stagger effect）
- **AND** 动画时长 0.6s

---

### 需求：搜索和筛选UI
博客页面的搜索和筛选组件需要现代化改造。

#### 场景：搜索栏
- **WHEN** 搜索栏渲染
- **THEN** 显示毛玻璃背景
- **AND** 边框发光效果
- **AND** 聚焦时边框光晕增强

#### 场景：下拉选择器
- **WHEN** 下拉菜单渲染
- **THEN** 显示与整体风格一致的毛玻璃效果
- **AND** 选项悬浮高亮

---

### 需求：代码块样式
文章中的代码块需要科技感样式。

#### 场景：代码块展示
- **WHEN** 代码块渲染
- **THEN** 显示深色背景
- **AND** 行号显示
- **AND** 语法高亮（使用 KaTeX 或自定义）
- **AND** 边框带有发光效果

## 视觉设计规范

### 色彩方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 深空背景 | `#0a0e27` | 主背景色 |
| 紫色星云 | `#1a0a2e` | 次要背景 |
| 渐变起点 | `#0f0c29` | 渐变开始 |
| 渐变中点 | `#302b63` | 渐变中间 |
| 渐变终点 | `#24243e` | 渐变结束 |
| 星光蓝 | `#4facfe` | 主色调 |
| 紫罗兰 | `#a855f7` | 辅助色 |
| 极光青 | `#00d4ff` | 强调色 |
| 文字主色 | `#ffffff` | 主要文字 |
| 文字次要 | `rgba(255,255,255,0.7)` | 次要文字 |

### 字体规范
- 主字体：`'Segoe UI', 'PingFang SC', 'Microsoft YaHei', sans-serif`
- 代码字体：`'Fira Code', 'Consolas', monospace`
- 标题字重：600-700
- 正文字重：400
- 标题字号：2.5rem - 3rem
- 正文字号：1rem - 1.1rem

### 间距规范
- 容器最大宽度：1000px
- 页面内边距：1rem - 3.5rem
- 卡片间距：2rem - 2.5rem
- 区块间距：1.5rem - 2.5rem
- 卡片内边距：2rem - 2.5rem

### 动画规范
- 默认过渡时长：0.3s
- 加载动画时长：0.6s
- 动画缓动函数：`cubic-bezier(0.4, 0, 0.2, 1)`
- 粒子数量（桌面端）：100-150
- 粒子数量（移动端）：50
- 流星出现频率：每 5-10 秒

### 毛玻璃效果规范
```css
background: rgba(255, 255, 255, 0.05);
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.1);
```

### 发光效果规范
```css
/* 默认状态 */
box-shadow: 0 0 20px rgba(79, 172, 254, 0.1);

/* 悬浮状态 */
box-shadow: 0 0 30px rgba(79, 172, 254, 0.3);

/* 强调状态 */
box-shadow: 0 0 40px rgba(168, 85, 247, 0.4);
```

## 技术实现指导

### CSS 变量定义
```css
:root {
  --bg-deep: #0a0e27;
  --bg-purple: #1a0a2e;
  --gradient-start: #0f0c29;
  --gradient-mid: #302b63;
  --gradient-end: #24243e;
  --accent-blue: #4facfe;
  --accent-purple: #a855f7;
  --accent-cyan: #00d4ff;
  --text-primary: #ffffff;
  --text-secondary: rgba(255, 255, 255, 0.7);
  --glass-bg: rgba(255, 255, 255, 0.05);
  --glass-border: rgba(255, 255, 255, 0.1);
}
```

### 粒子系统实现要点
1. 使用 Canvas API 绘制
2. 粒子属性：位置、大小、透明度、速度
3. 流星效果：带尾巴的快速移动粒子
4. 鼠标交互：检测鼠标位置，影响附近粒子

### 打字机效果实现
1. 将标题文本拆分为字符数组
2. 使用 setInterval 逐字添加
3. CSS 控制光标闪烁动画

- Chrome 80+

## 性能优化

### 动画性能
- 使用 transform 和 opacity
- 避免触发布局重排的属性
- 使用 will-change 提示浏览器

### 粒子系统优化
- 控制粒子数量
- 使用 requestAnimationFrame
- 低性能设备降级
